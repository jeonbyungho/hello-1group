## 브런치 생성

```
> git branch testing
```

## 브런치 이동
```
> git checkout testing
Switched to branch 'testing'
```

## 다른 브런치와 병합
```
> git merge testing
Updating ...
```

## 브런치 목록 전체 보기
```
> git branch
* main
  testing
```
